# ejdb2-rs

EJDB2 rust binding

## Usage

Add ejdb2-rs as dependency in your `Cargo.toml`

```toml
[dependencies]
ejdb2={git=https://github.com/Joylei/ejdb2-rs.git}
```
Please see test cases for usage details.
