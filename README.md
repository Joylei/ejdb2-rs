# ejdb2-rs

EJDB2 rust binding

## How to use

Add ejdb2-rs as your project dependency

Cargo.toml
```toml
[dependencies]
ejdb2={git=https://github.com/Joylei/ejdb2-rs.git}
```
Please see test cases for usage details.
